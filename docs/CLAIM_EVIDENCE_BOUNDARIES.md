# Claim-Evidence Boundaries

| Claim | Direct evidence | Boundary |
|---|---|---|
| Released validation and test sets share recording sources | Identifier audit | No score-inflation magnitude is inferred |
| MuQ is the stable unadjusted reference | Pooled OOF UAR, macro-F1, accuracy | Not evidence of singer-disjoint transfer |
| Class-bias adjustment raises UAR | Multiple encoders and policy weights | Not a uniformly superior classifier |
| Adjusted fusion beats adjusted MuQ | Not resolved by paired intervals | No fusion superiority claim |
| Explicit descriptors add held-aria information | Not supported by the permutation audit | Descriptor names are not validated measurements |
| Results transfer across arias | Six SVQTD tenor arias | No claim for other voice types, styles, or languages |
| Errors reflect model limitations | Partly supported | Rater-level data are unavailable, so annotation disagreement cannot be separated |
